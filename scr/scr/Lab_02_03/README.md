## Базовый класс Student
 Модель сущности «студент», содержащая общие атрибуты и методы для всех типов студентов.
### Атрибуты:
- _fio — ФИО
- _birthdate — дата рождения
- _group — группа
- _course — курс (вычисляется автоматически)
- gpa — средний балл (GPA)
## Дочерний класс BachelorStudent (бакалавр)
### Добавленные атрибуты:
- specialization — специализация (строка)
- has_thesis — наличие защищённой дипломной работы (bool)
### Переопределённые методы:
- get_scholarship_amount() — базовая стипендия + 500 руб., если диплом защищён
- __str__() — добавлен вывод специализации и статуса диплома
### Новый метод:
- defend_thesis() — защита диплома (меняет has_thesis)
## Дочерний класс MasterStudent (магистр)
### Добавленные атрибуты:
- research_area — область исследований (строка)
- supervisor — научный руководитель (строка)
- papers_count — количество публикаций (целое)
## Переопределённые методы:
- get_scholarship_amount() — базовая стипендия + 300 руб. за каждую публикацию
- __str__() — добавлен вывод области исследований, руководителя и количества публикаций
## Новый метод:
- publish_paper(conference) — публикация статьи (увеличивает papers_count)
