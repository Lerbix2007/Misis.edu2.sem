## Стратегии фильтрации (предикаты)
- Функции, принимающие объект Student и возвращающие bool:

- is_active(student) – возвращает True, если студент активен (не отчислен).

- is_excellent(student) – возвращает True, если средний балл студента ≥ 4.8.

## Стратегии сортировки (ключи сравнения)
- Функции, извлекающие из объекта Student значение для сравнения:

- by_fio(student) – возвращает ФИО для лексикографического сравнения.

- by_gpa(student) – возвращает средний балл для числового сравнения.

## Стратегии преобразования (apply/отображения)
- Функции или callable-объекты, преобразующие Student в другой формат:

- student_to_short_string(student) -> str – возвращает короткую строку: "Иванов И. (4.85)".

- ShortInfoStrategy – callable-класс, возвращает строку с ФИО и группой.

- GpaBonusStrategy – callable-класс, начисляет бонус к GPA (принимает коэффициент в __init__).

## Классы, использующие стратегии (делегаты)
- StudentCollection (коллекция студентов)
Реализует методы, принимающие стратегии как аргументы:

- filter_by(predicate: Callable[[Student], bool]) -> StudentCollection
Принимает функцию-предикат и возвращает новую коллекцию с отфильтрованными студентами.

- sort_by(key_func: Callable[[Student], Any], reverse: bool = False) -> StudentCollection
Принимает функцию извлечения ключа и флаг порядка, возвращает отсортированную коллекцию.

- apply(transform: Callable[[Student], Any]) -> list[Any]
Принимает функцию или callable-объект для преобразования каждого студента.

### Student (базовый класс)
Не реализует методы сравнения напрямую. Вместо этого стратегии сравнения передаются извне.

### BachelorStudent, MasterStudent (наследники Student)
Могут использоваться с теми же стратегиями, что демонстрирует полиморфизм без изменения кода стратегий.

- Универсальные функции (делегаты), работающие через стратегии
filter_students(students: list[Student], predicate: Callable[[Student], bool]) -> list[Student]
Принимает список студентов и функцию-предикат, возвращает отфильтрованный список.

Проверка через callable() перед вызовом переданной стратегии.
