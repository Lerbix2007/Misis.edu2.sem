## Student

&nbsp;&nbsp;&nbsp;&nbsp;В работе реализован пользовательский класс **Student**, который моделирует студента в образовательной системе. Класс содержит атрибуты объекта: ФИО, дату рождения, группу и средний балл. Также реализовано вычисление возраста и курса студента на основе даты рождения.

### Атрибуты класса

&nbsp;&nbsp;&nbsp;&nbsp;**`university_name`** — название университета (общее для всех студентов)  
&nbsp;&nbsp;&nbsp;&nbsp;**`total_students`** — счетчик созданных студентов

### Закрытые поля:

&nbsp;&nbsp;&nbsp;&nbsp;*   `_fio` — ФИО студента  
&nbsp;&nbsp;&nbsp;&nbsp;*   `_birthdate` — дата рождения  
&nbsp;&nbsp;&nbsp;&nbsp;*   `_group` — учебная группа  
&nbsp;&nbsp;&nbsp;&nbsp;*   `_gpa` — средний балл (GPA)  
&nbsp;&nbsp;&nbsp;&nbsp;*   `_course` — курс обучения (вычисляется)

### Свойства (`@property`):

&nbsp;&nbsp;&nbsp;&nbsp;*   **`fio`** — чтение ФИО  
&nbsp;&nbsp;&nbsp;&nbsp;*   **`birthdate`** — чтение даты рождения в отформатированном виде  
&nbsp;&nbsp;&nbsp;&nbsp;*   **`birthdate_raw`** — чтение даты рождения как объекта (для вычислений)  
&nbsp;&nbsp;&nbsp;&nbsp;*   **`group`** — чтение и запись группы  
&nbsp;&nbsp;&nbsp;&nbsp;*   **`gpa`** — чтение и запись среднего балла  
&nbsp;&nbsp;&nbsp;&nbsp;*   **`course`** — чтение курса (вычисляется автоматически)

### Магические методы:

&nbsp;&nbsp;&nbsp;&nbsp;*   **`__str__`** — удобный вывод информации о студенте  
&nbsp;&nbsp;&nbsp;&nbsp;*   **`__repr__`** — техническое представление объекта для разработчиков  
&nbsp;&nbsp;&nbsp;&nbsp;*   **`__eq__`** — сравнение студентов по ФИО и дате рождения

### Бизнес-методы:

&nbsp;&nbsp;&nbsp;&nbsp;*   **`get_age()`** — вычислить возраст студента  
&nbsp;&nbsp;&nbsp;&nbsp;*   **`get_gpa_grade()`** — перевести GPA в буквенную оценку  
&nbsp;&nbsp;&nbsp;&nbsp;*   **`get_student_card()`** — сгенерировать номер студенческого билета
